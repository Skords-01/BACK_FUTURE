# Гайд для контенту

## Тон

- **Науково-популярний.** Не конспект, не Вікіпедія, не пост у телеграмі. Уявляй розумного, але далекого від теми друга, якому 30+.
- **Український, нейтральний.** Не використовуємо суржик, кальки з російської, надмірний пафос.
- **Без політичної агітації.** Особливо в історії. Спираємось на офіційні документи.

## Структура факту

```
title       — короткий заголовок, до 80 символів. Без знаків пунктуації в кінці.
short       — 1–2 речення, 20–280 символів. Перше враження. Має містити «що нового».
before      — опц., 10–280 символів. Що було в підручнику / як вчили раніше.
after       — опц., 10–280 символів. Що тепер — лаконічно, без розгорнутих пояснень.
body (md)   — 3–5 абзаців. Може мати заголовки h2 (`##`), списки, цитати.
```

### `before` / `after`

Це візитка факту: на картці одразу під `short` виводиться блок «Було → Стало». Поля
опціональні — можна заповнити одне з двох (наприклад, тільки `after` для нового
явища, якого в шкільній програмі взагалі не було). Тримай їх **дуже короткими**:
одна-дві фрази, без markdown, без посилань. Розгорнуті пояснення — у `body`.

Заповнюй `before`/`after` **там, де є чітке протиставлення**: класифікації
(Плутон: 9 → 8 планет), процедури (CRISPR: дороге та рідке → рутина), стан
об'єкта (Арал: 68 000 км² → ~10%). Якщо «до/після» виглядає штучно — краще
лишити поля порожніми.

### `region`

Опціональне поле, значення: `"world"` або `"ukraine"`. Це формальна заміна
тегу `ukrainian-contribution`, який ми колись клеїли на українські факти.

- `region: "ukraine"` — український внесок або факт, що відбувся переважно в
  Україні (Каденюк-1997, Чурюмов/Розетта-2014, Голодомор як геноцид, мовний
  закон, харківські сцинтилятори для CERN тощо). На картці з'являється бейдж
  «🇺🇦 Україна».
- `region: "world"` — світова подія без сильної локальної прив'язки (бозон Хіггса,
  СМЕРШ останнього мамонта, Plut → 8 planet). Бейджа немає; це дефолтний намір,
  тож поле можна просто не вказувати.

Не пиши `region` як «світ і Україна». Якщо у факту дійсно дві сторони — обирай
ту, яка дає сильніший сигнал у стрічці (ми робимо чек-бокс «тільки про Україну»,
а не градацію).

## Шаблон тіла

```
[1 абзац: контекст — як було колись, що вчили в школі]
[1 абзац: ключова подія / відкриття / зміна, з датою]
[1 абзац: деталі — як це працює / чому важливо]
[опц. 1 абзац: наслідки / відкриті питання]
[опц. цитата `>`: запам'ятовувана думка]
```

## Джерела

- Мінімум **одне** першоджерело (academic paper, IAU resolution, FDA, нацакадемія, держархів).
- Уникати: рандомні блоги, агрегатори новин, Wikipedia як єдине джерело.
- Wikipedia можна як «огляд», але не як єдину опору.

## Що рахується «оновленням»

- Нове відкриття / винахід (експеримент, винахід технології).
- Уточнення / спростування попереднього погляду.
- Зміна офіційної класифікації (як з Плутоном).
- Зміна на політичній карті (нова країна, перейменування).
- Розсекречення архівів / нова історична оцінка з юридичною ваги.

## Чого уникати

- **Гарячих новин** (молодші 6 місяців, поки не закріпились).
- **Спекулятивних теорій** без експериментального підтвердження.
- **Особистих думок** автора факту.
- **Russia-centric framing** (наприклад, коли українська історія описується «ззовні»).

## Чутливі формулювання

### Російсько-українська війна

Дотримуємось офіційної української термінології:

- **Війна почалась 20 лютого 2014 року** з російської окупації Криму та частини Донбасу.
- **24 лютого 2022 року** — початок **повномасштабного вторгнення** Росії в Україну (це новий, найгарячіший етап тієї ж війни, а не «нова війна»).
- **Не пишемо** «війна почалась у 2022-му», «війна 2022 року», «за рік війни» (без уточнення).
- **Пишемо** «російсько-українська війна (з 2014)», «повномасштабне вторгнення (з 24 лютого 2022)», «з початку повномасштабної фази».
- **Уникати** евфемізмів типу «конфлікт», «спецоперація», «події в Україні».

### Голодомор

Завжди як **геноцид українського народу 1932–1933 років**. Нагадуємо про юридичні визнання (закон України 2006, резолюція Європарламенту 2022 тощо).

### Окупація / анексія

«Анексія Криму» — цитуємо так лише в лапках або з уточненням «незаконна спроба анексії». В нормальному тексті — **окупація Криму Росією** з 2014 року.

### Топоніми

- Користуємось українською транслітерацією: **Київ** (Kyiv), **Одеса** (Odesa), **Львів** (Lviv), **Харків** (Kharkiv), **Чорнобиль** (Chornobyl).
- Не «Kiev», «Odessa», «Chernobyl», «Lvov».
- В іноземних URL/назвах документів зберігаємо оригінал ([Reuters article on Kyiv]), але в основному тексті — український варіант.

## Перевірка

Перед мерджем PR з новим фактом має пройти:

1. `npm run build` — Zod-валідація frontmatter.
2. `npm run validate:content` — додаткові quality gates: джерела, чернетки, sensitive wording.
3. `npm run lint:md` — Markdown-структура.
4. `npm run coverage:content` — перевірити, чи не посилюємо прогалини по ерах/предметах.
5. Хоча б одне джерело — клікабельне й живе.
6. Рев'ю автора проєкту: тон, формулювання, відсутність помилок.
