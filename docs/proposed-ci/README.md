# Proposed CI

Файл `ci.yml` тут — це готовий GitHub Actions workflow. Я не зміг змерджити його в `.github/workflows/` напряму, бо OAuth-токен Devin не має `workflow`-скоупу.

**Як додати:** скопіюй `ci.yml` у `.github/workflows/ci.yml` локально і запуш зі свого облікового запису. Або скопіюй вміст і встав через UI GitHub → Add file → Create new file.
